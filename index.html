Some rules to follow:

  • Never work on main
  • Never work on develop
  • What's not your responsibility, don't touch it!
  • When you create a branch, do a PUSH to that branch to create it on GitHub
  • Always push to your working branch
  • Working branches must have descriptive names
  • Commit messages must be descriptive
  • Before doing a push, do a pull from develop
  • Before starting to work on your branch, do a pull from develop
  • Every time someone modifies develop, do a PULL!
